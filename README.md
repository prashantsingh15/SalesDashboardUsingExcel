## Interactive Excel Sales Dashboard

### Overview
This project presents an interactive sales dashboard built in Microsoft Excel to analyze sales performance, profit trends, and customer behavior across multiple years. The dashboard enables dynamic filtering by product category and year to support data-driven decision-making.

### Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Excel Maps
- Excel Tables

### Dataset
- Sales transaction data from 2014 to 2017
- Includes category, sub-category, state, customer, sales, profit, and order details

### Data Preparation
- Cleaned raw sales data and removed inconsistencies
- Structured data into Excel Tables for dynamic updates
- Created calculated fields for profit and customer metrics

### Dashboard Features
- **Category & Year slicers** for interactive filtering
- **Sales by Category** bar chart highlighting top-performing products
- **Profit Gained Over Time** line chart segmented by category
- **Sales by State** map visualization for geographic insights
- **Customer Count by Year** comparison
- **Top 5 Customers Making Profit** pie chart
- **Monthly Sales Trend** area chart to identify seasonality

### Key Insights
- Technology and Office Supplies show consistent profit growth over time
- A small group of customers contributes significantly to total profit
- Sales demonstrate strong seasonality, with peaks toward year-end
- Certain states outperform others in overall sales contribution

### How to Use
1. Download the `Sales_Dashboard.xlsx` file
2. Open it in Microsoft Excel
3. Click **Refresh All** to update visuals
4. Use slicers to explore sales trends by category and year
